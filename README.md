# hey, I'm Nishant 👋

I started out wanting to build bridges. Somewhere along the way I fell for the software side instead. I still care about the same problem, really: how do you actually get the thing built. Code just turned out to be a much better tool for chasing that than I expected.

### Right now

- 🏗️ Running **[InfraHacks](https://github.com/infrahacks2027)**. I love hackathons, and I kept looking for a flagship one built around infrastructure and couldn't find one, so I started one.
- 🚀 Local Lead for **NASA Space Apps Challenge, London ON** (Nov 14–15, 2026). Basically a mini space hackathon for my city.
- ⚡ Building a custom motor controller board from scratch (STM32, closed-loop FOC, the whole thing), mostly because I wanted to actually understand what's happening under the hood of a motor instead of trusting a library to handle it.

Still figuring out exactly where "started in civil, ended up in code" lands long-term. I just know it's somewhere that lets me keep building things that touch the real world, not just the screen.
